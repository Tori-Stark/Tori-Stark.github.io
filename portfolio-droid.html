Droid Cafe

About

  • Category: Android Development
  • Language: Java
  • Project Duration: 1 Month
  • Project URL: GitHub
Description

Droid CAFE App is a delightful application that showcases the use of images as buttons and a floating action button to trigger intents and launch a second activity. With its visually appealing interface and seamless functionality, this app allows users to explore and discover a wide variety of recipes. From appetizers to desserts, you can browse through a collection of mouth-watering dishes, each accompanied by a tantalizing image. Simply tap on the desired recipe image to view detailed instructions, ingredients, and cooking tips. The floating action button provides a convenient way to add your favorite recipes to a personal collection or share them with friends.

Features
  • Use of images as buttons
  • Use of floating action button to trigger intents and launch a second activity
  • Use of RecyclerView to display a grid of images
  • Use of CardView to display recipe information
  • Use of implicit intents to launch a browser and display a location on a map
  • Use of ShareCompat.IntentBuilder to create an action for sharing recipe text
  • Use of Picasso library to load images
  • Use of Glide library to load images
